SETUP REQUIRED

Connect Firebase

The website is complete, but public edits need a shared database. Follow the included README to connect your free Firebase project.

  1. Create a Firebase project and Firestore database.
  2. Enable Email/Password authentication.
  3. Create your private administrator account.
  4. Paste the Firebase web configuration into firebase-config.js.
  5. Publish the included Firestore security rules using your Firebase UID.
